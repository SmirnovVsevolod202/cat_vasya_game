# cat_vasya_game
Ролевая игра с пошаговой боевкой от кота Василия

## Описание

Проект реализует систему классов для RPG с:
Персонажами разных классов (воин, маг, охотник)
Монстрами с уникальными характеристиками
Системой заклинаний и маны
Пошаговой боевой механикой

## Модули

### Module 1
Базовая архитектура:
Абстрактный класс `Unit` с шестью характеристиками
Класс `Character` для персонажей игроков
Класс `Monster` для противников
Абстрактные методы расчёта здоровья, урона и защиты

### Module 2
Система классов персонажей:
Единый класс `Character` с атрибутом `character_class`
Поддержка трёх классов: warrior, mage, hunter
Уникальные формулы расчёта урона и защиты для каждого класса

### Module 3
Система магии:
Абстрактный класс `Spell` для заклинаний
Конкретные заклинания: Fireball, IceLance, LightningBolt
Система маны с разными формулами для классов
Методы добавления и применения заклинаний
